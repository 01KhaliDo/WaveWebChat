# WaveWebChat

En enkel och smidig webbapplikation för privata realtidskonversationer. Byggd med Node.js och Socket.io. 
## Live Demo

Prova applikationen direkt här:  
👉 **[https://wavewebchat.vercel.app/](https://wavewebchat.vercel.app/)**

> **Obs:** Servern stängs av automatiskt efter inaktivitet. Därför kan det ta **några sekunder** att starta upp igen första gången du besöker länken.

## Funktioner

- **Privata Chattar:** Välj en användare från listan för att starta en privat chatt.
- **Skrivindikatorer:** Se när någon skriver ett meddelande ("TYPING...").
- **Ljudnotiser:** Få ljudsignaler vid inkommande meddelande.
- **Användarstatus:** Se vem som är online och antal olästa meddelanden.

## Teknologier

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** HTML, CSS, JavaScript

## Hur du använder chatten

1. **Gå med:** Öppna länken och välj ett användarnamn för att logga in.
2. **Se användare:** I listan till vänster ser du andra som är anslutna just nu.
3. **Starta chatt:** Klicka på en person i listan du vill prata med.
4. **Interagera:** 
   - Skriv ditt meddelande i rutan längst ner och tryck på "Skicka".
   - Se när motparten skriver.
   - Hör ett ljud när du får svar.

# WaveWebChat (English)

A simple and smooth web application for private real-time conversations. Built with Node.js and Socket.io for seamless communication.

## Live Demo

Try the application directly here:  
👉 **[https://wavewebchat.vercel.app/](https://wavewebchat.vercel.app/)**

> **Note:** The server shuts down automatically after inactivity. Therefore, it may take **a few seconds** to start up again the first time you visit the link.

## Features

- **Private Chats:** Select a user from the list to start a private chat.
- **Typing Indicators:** See when someone is writing a message ("typing...").
- **Sound Notifications:** Receive audio alerts for incoming messages.
- **User Status:** See who is online and the number of unread messages.

## Technologies

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** HTML, CSS, JavaScript

## How to use the chat

1. **Join:** Open the link and choose a username to log in.
2. **View users:** In the list on the left, you can see others who are currently connected.
3. **Start chat:** Click on a person in the list you want to talk to.
4. **Interact:** 
   - Write your message in the box at the bottom and press "Send".
   - See when the other person is typing.
   - Hear a sound when you receive a reply.
