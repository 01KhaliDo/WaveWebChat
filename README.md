# WaveWebChat (English)

A simple and smooth web application for private real-time conversations. Built with Node.js and Socket.io for seamless communication.

## Live Demo

Try the application directly here:  
游녤 **[https://wavewebchat.vercel.app/](https://wavewebchat.vercel.app/)**

> **Note:** The server shuts down automatically after inactivity. Therefore, it may take **a few seconds** to start up again the first time you visit the link.

## Features

- **Private Chats:** Select a user from the list to start a private chat.
- **Typing Indicators:** See when someone is writing a message ("typing...").
- **Sound Notifications:** Receive audio alerts for incoming messages.
- **User Status:** See who is online and the number of unread messages.

## Technologies

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** HTML5, CSS3, JavaScript

## How to use the chat

1. **Join:** Open the link and choose a username to log in.
2. **View users:** In the list on the left, you can see others who are currently connected.
3. **Start chat:** Click on a person in the list you want to talk to.
4. **Interact:** 
   - Write your message in the box at the bottom and press "Send".
   - See when the other person is typing.
   - Hear a sound when you receive a reply.
